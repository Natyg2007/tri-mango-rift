# Mango Phase Circuit

Mango Phase Circuit is a standalone 2D browser game prototype built with HTML,
CSS, and JavaScript Canvas.

## Controls

- Move: WASD or arrow keys
- Shift phase: Space
- Pick phase directly: 1, 2, 3

## Work Sample Focus

This is intentionally different from Tri-Mango Rift 3D. It demonstrates a
2D-style game loop, random arena generation, phase-matched teleport pads,
wrong-color spinning hazards, mango collection, score/time penalties, wormhole
stage transitions, and an info panel that returns after the final win state.
